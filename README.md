---

## 📋 Descripción

Repositorio que simula casos reales de tickets de soporte técnico,  
documentando el proceso completo: recepción, diagnóstico, resolución y cierre.  
Basado en buenas prácticas de **ITIL** para gestión de incidencias.

---

## 📁 Casos documentados

| Ticket | Categoría | Prioridad | Estado |
|--------|-----------|-----------|--------|
| `TKT-001` | Impresora sin respuesta | Media | ✅ Resuelto |
| `TKT-002` | Sin acceso a Internet | Alta | ✅ Resuelto |
| `TKT-003` | Olvido de contraseña | Baja | ✅ Resuelto |
| `TKT-004` | PC lenta / alto uso de CPU | Media | ✅ Resuelto |

---

## 🎯 Campos de cada ticket

- **ID y fecha** de apertura
- **Usuario** afectado
- **Descripción** del problema
- **Diagnóstico** realizado
- **Solución aplicada**
- **Tiempo de resolución**
- **Estado** del ticket
